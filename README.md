<h1>DC FeaturedBoxes for Joomla</h1>
<p>DC FeaturedBoxes is a powerful and flexible Joomla module designed to create modern, responsive feature/service box sections with advanced layout control and clean architecture.</p>
<p>It allows administrators to build visually dynamic content blocks without touching code, while maintaining full developer-level extensibility.</p>

<h2>🏗 Module Architecture</h2>
<p>The module is built with clean separation of concerns:</p>
<h3><code>mod_dc_featuredboxes.xml</code></h3>
<p>Defines configuration parameters and admin options.</p>
<h3><code>default.php</code></h3>
<p>Handles rendering logic, conditional classes, and layout behavior.</p>
<h3>CSS Styling</h3>
<p>Based on modular classes such as:</p>
<ul>
<li>
<p><code>dc-featuredboxes-wrapper</code></p>
</li>
<li>
<p><code>dc-fbox-cols-X</code></p>
</li>
<li>
<p><code>dc-fbox-every-second-lower</code></p>
</li>
<li>
<p><code>dc-fbox-grid-margin-bottom</code></p>
</li>
</ul>
<p>This approach ensures maintainability, scalability, and ease of customization.</p>
